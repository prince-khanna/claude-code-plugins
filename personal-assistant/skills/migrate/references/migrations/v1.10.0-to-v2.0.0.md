# Migration: v1.10.0 → v2.0.0

## Summary
- Replaces UserPromptSubmit hook with native `~/.claude/rules/imli-core.md`
- ~95% token savings over typical sessions
- Adds SessionStart hooks for trigger checks and compaction resilience
- Adds skills: `/sync-context`, `/context-health`, `/evolve`

## Breaking Changes
- `UserPromptSubmit` hook removed — context delivery is now via native rules
- `load_context_system.py` removed — replaced by `imli-core.md`
- `update_context_on_stop.py` removed — context updates are explicit via `/update-context`

## Steps

### 1. Create rules directory
**Check:** `[ -d ~/.claude/rules ]`
**Action:** `mkdir -p ~/.claude/rules`
**Verify:** `[ -d ~/.claude/rules ]`

### 2. Generate imli-core.md
**Check:** `[ -f ~/.claude/rules/imli-core.md ]`
**Action:** `uv run python ${CLAUDE_PLUGIN_ROOT}/skills/sync-context/scripts/sync_context.py`
**Verify:** `[ -f ~/.claude/rules/imli-core.md ] && [ -s ~/.claude/rules/imli-core.md ]`

### 3. Update instruction files
**Check:** `grep -q "native rules" ~/.claude/.context/CLAUDE.md 2>/dev/null`
**Action:** `cp ${CLAUDE_PLUGIN_ROOT}/context-template/CLAUDE.md ~/.claude/.context/CLAUDE.md && cp ${CLAUDE_PLUGIN_ROOT}/context-template/context-update.md ~/.claude/.context/context-update.md`
**Verify:** `[ -f ~/.claude/.context/CLAUDE.md ]`

### 4. Ensure relationships.md exists
**Check:** `[ -f ~/.claude/.context/core/relationships.md ]`
**Action:** `cp ${CLAUDE_PLUGIN_ROOT}/context-template/core/relationships.md ~/.claude/.context/core/relationships.md`
**Verify:** `[ -f ~/.claude/.context/core/relationships.md ]`

### 5. Ensure triggers.md exists
**Check:** `[ -f ~/.claude/.context/core/triggers.md ]`
**Action:** `cp ${CLAUDE_PLUGIN_ROOT}/context-template/core/triggers.md ~/.claude/.context/core/triggers.md`
**Verify:** `[ -f ~/.claude/.context/core/triggers.md ]`

### 6. Ensure journal.md exists
**Check:** `[ -f ~/.claude/.context/core/journal.md ]`

**Verify:** `[ -f ~/.claude/.context/core/journal.md ]`

### 7. Ensure improvements.md exists
**Check:** `[ -f ~/.claude/.context/core/improvements.md ]`
**Action:** `cp ${CLAUDE_PLUGIN_ROOT}/context-template/core/improvements.md ~/.claude/.context/core/improvements.md`
**Verify:** `[ -f ~/.claude/.context/core/improvements.md ]`

### 8. Check for duplicate notification hooks
**Check:** `python3 -c "import json; d=json.load(open('$HOME/.claude/settings.json')); exit(0 if not d.get('hooks',{}).get('Stop') and not d.get('hooks',{}).get('Notification') else 1)" 2>/dev/null`
**Action:** `echo "MANUAL: Check ~/.claude/settings.json for Stop/Notification hooks that duplicate the plugin hooks. Remove duplicates from settings.json."`
**Verify:** `echo "User should verify no duplicate notification sounds"`

### 9. Clean up old directories
**Check:** `[ ! -d ~/.claude/.context/projects ] && [ ! -d ~/.claude/.context/session ]`
**Action:** `rmdir ~/.claude/.context/projects/ 2>/dev/null; rmdir ~/.claude/.context/session/ 2>/dev/null; true`
**Verify:** `[ ! -d ~/.claude/.context/projects ] && [ ! -d ~/.claude/.context/session ]`
